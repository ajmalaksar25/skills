# skills

My personal agent skills for Claude Code.

## Skills

| Skill | What it does |
|-------|--------------|
| [handoff-with-prompt](handoff-with-prompt/) | Condense a long conversation into a durable handoff document plus a separate paste-ready handoff prompt, so a fresh session resumes work without losing detail to compaction. |

## Install

Symlink a skill into your skills directory:

```sh
ln -s "$PWD/handoff-with-prompt" ~/.claude/skills/handoff-with-prompt
```
